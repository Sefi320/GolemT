## Reactive Data Contract

Every module server receives a single reactive argument: `app_data`.

`app_data()` returns a named list:
- `app_data()$futures` — the full `RTL::dflong` data frame
- `app_data()$cmt` — CMT yields from `fetch_cmt()`
- `app_data()$eia` — PADD data from `fetch_eia()`

Modules never fetch their own data. They call `filter_futures(app_data()$futures, "CL")` etc. to get what they need.

- Modify: `R/app_ui.R` (add CL tab)

**What mod_cl does:**
The CL tab follows the standard structure. Each output is a separate `render*` function inside `mod_cl_server`.

**Outputs to build:**
1. `price_chart` — `renderPlotly`: CL01 price over time with vertical lines at event marker dates (from `utils_events.R`), tooltip showing event name on hover

5. `vol_surface` — `renderPlotly`: 3D surface using `calc_rolling_vol()` output for all CL contracts; x=contract month, y=date, z=rolling_vol
6. `garch_chart` — `renderPlotly`: GARCH conditional vol from `calc_garch()` for the user-selected contract

**Interactive inputs:**
- `contract_select` — selectInput of CL contracts (CL01–CL36) for GARCH
- `rolling_window` — sliderInput (30, 60, 90, 252) affecting rolling vol and surface

## Task 7: CMT Module

**Files:**
- Create: `R/mod_cmt.R`
- Modify: `R/app_ui.R`, `R/app_server.R`

**What mod_cmt does:**

This tab uses `app_data()$cmt` from FRED. Structure is different from commodity tabs:

1. `curve_animation` — `renderPlotly`: animated yield curve (x = maturity label, y = yield %), one frame per month. Shows curve morphing from normal (2018) → inverted (2022-2023) → re-steepening (2024). Play/pause + timeline slider.

2. `narrative` — `renderUI`: static text explaining normal/inverted/flat curve shapes and what they signal about economic expectations. Connection to commodity demand channels.

3. `rate_cycle_chart` — `renderPlotly`: time series of 3-month yield and 10-year yield on same chart, with Fed rate decision event markers. Replaces the seasonality heatmap for CMT.

4. `rolling_vol_chart` — `renderPlotly`: rolling volatility for short end (3M), belly (5Y), and long end (30Y) plotted together. Shows event-driven vs persistent volatility.

5. `cmt_brn_chart` — `renderPlotly`: 10Y CMT yield vs BRN01 price on dual axis, showing rate hike cycle and BRN price pressure relationship.
